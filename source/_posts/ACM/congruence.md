# 定理

1. $r \bmod m = \{ r + km \mid k \in \mathbb{Z} \}$

2. 设 $m = m_1 m_2 \dots m_k$，并令
   $$ x = x^{(1)} + m_1 x^{(2)} + m_1 m_2 x^{(3)} + \dots + m_1 m_2 \dots m_{k-1} x^{(k)}. $$
   当 $x^{(j)}\ (1 \le j \le k)$ 分别遍历 $m_j$ 的完全剩余系时，$x$ 遍历 $m$ 的完全剩余系。

3. 若 $a_1, a_2, \dots, a_k$ 满足 $(a_j, m_j) = 1\ (1 \le j \le k)$，并令
   $$ x = a_1 x^{(1)} + m_1 a_2 x^{(2)} + m_1 m_2 a_3 x^{(3)} + \dots + m_1 m_2 \dots m_{k-1} a_k x^{(k)}, $$
   则当 $x^{(j)}\ (1 \le j \le k)$ 分别遍历 $m_j$ 的完全剩余系时，$x$ 遍历 $m$ 的完全剩余系。

4. 设 $m = m_1 m_2 \dots m_k$，且 $m_1$ 与 $m$ 有相同的素因数。当 $x^{(1)}$ 遍历 $m_1$ 的完全剩余系，$x^{(j)}\ (2 \le j \le k)$ 分别遍历 $m_j$ 的完全剩余系时，
   $$ x = x^{(1)} + m_1 x^{(2)} + m_1 m_2 x^{(3)} + \dots + m_1 m_2 \dots m_{k-1} x^{(k)} $$
   则 $x$ 遍历 $m$ 的完全剩余系。

5. 设 $m = m_1 m_2$，且 $(m_1, m_2) = 1$，
   $$ x = m_2 x^{(1)} + m_1 x^{(2)}. $$
   则 $x$ 遍历 $m$ 的完全（既约）剩余系当且仅当 $x^{(1)}$ 与 $x^{(2)}$ 分别遍历 $m_1$ 与 $m_2$ 的完全剩余系。
